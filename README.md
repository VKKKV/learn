# Learn

A comprehensive collection of learning projects, exercises, and experiments across a wide range of programming languages and technologies.

## 📁 Repository Structure

### 💻 Low-Level & Systems
- **[Assembly](./assemble/)**: x86-64 assembly experiments using AT&T syntax.
- **[C](./c/)**: Fundamental C programming projects.
- **[Rust](./rust/)**: Various Rust projects including CLI tools (`minigrep`) and language fundamentals.
- **[QEMU](./qemu/)**: Virtualization configurations and environment setups.

### 🌐 Web & Backend
- **[Go](./go/)**: Web development using the Gin framework and a URL shortener project.
- **[JavaScript](./javascript/)**: Web-based interactions and experiments.
- **[Solidity](./solidity/)**: Ethereum smart contract development (FundMe, SimpleStorage).

### 🐍 Python & Data Science
- **[Python](./python/)**: Security-focused scripts (CryptoHack), network tools (Scapy), and general utilities.
- **[Matplotlib](./matplotlib/)**: Data visualization examples.

### 🎮 Graphics & Multimedia
- **[GLSL](./glsl/)**: Shader programming for graphics and compute tasks.
- **[Strudel](./strudel/)**: Live coding music experiments.
- **[Forge](./forge/)**: Minecraft mod development using Java and Gradle.

### 🧩 Algorithms & Problem Solving
- **[LeetCode](./leetCode/)**: Solutions to algorithmic challenges in C++, Python, and C.

### 📝 Documentation & Typesetting
- **[Cheatsheets](./cheatsheet/)**: Quick reference guides for Bash and Pwntools.
- **[LaTeX](./latex/)** & **[Typst](./typst/)**: Document preparation and typesetting experiments.

### 🧪 Miscellaneous
- **[Java](./java/)**: Multithreading experiments.
- **[LISP](./LISP/)** & **[Lua](./lua/)**: Basic language exploration scripts.
